# 012_何もできない日の自己管理  

何もできない日は、  
壊れたんじゃない。  

体も心も、  
ゆっくり冷ましているだけ。  

世界を読もうとしても、  
頭の中が止まらない日がある。  
そんなときは、  
考えるよりも、  
感じる範囲を小さくする。  

朝、目を開けて、  
カーテンのすき間から光を見てみる。  
それだけでいい。  

今日はそれが、  
「できたこと」になる。  

何かを決めようとしなくていい。  
元気が出てからでいい。  
焦らなくても、  
世界の方から、少しずつ近づいてくる。  

動けない自分を責めずに、  
「今は止まっているんだな」と読む。  
それが、  
自分を整えるいちばん小さな方法。  

自己管理とは、  
自分をコントロールすることじゃない。  
自分の状態を読んであげること。  

天気を見るように、  
心の調子を眺めてみる。  
雨の日も、晴れの日もある。  
それでいい。  

何もできない日は、  
静かに自分を読む日。  
それも、ちゃんとした一日。  

そのうち、身体が、  
「もう動いていいよ」と言ってくれるから。

# 013_少し動ける日の自己整え ― 世界に戻るリハビリ  

少し体が動く日。  
でも、心がまだ追いつかない日。  

そんな日こそ、  
世界に戻るリハビリを始める。  

やることは、たったひとつでいい。  

ゴミを1個捨てる。  
食器を1枚洗う。  
シャワーを浴びてみる。  
スクワットを1回だけやってみる。  

どれも、  
“世界に触れる”ための小さな動き。  

動けるからといって、  
全部やろうとしなくていい。  
リハビリとは、  
力を使うことではなく、感覚を取り戻すこと。  

行動の大きさよりも、  
「やってみた」と思える感覚を大事にする。  
それが、  
明日へつながる“再起動”のサインになる。  

世界は、少しずつ戻ってくる。  
体と心がそろって歩き出すその日まで、  
ゆっくり呼吸を合わせていけばいい。

# 014_ふつうの日の自己整え ― 平和を維持する力  

何も特別じゃない日。  
天気も心も、まあまあ。  

そんな「ふつうの日」こそ、  
整える力を試す日。  

朝の支度を、少しだけ丁寧にしてみる。  
お茶を入れる。  
洗面台を整える。  
カーテンを開けて、風を入れる。  

ほんの数分でも、  
「整える時間」があると、  
心は一日中、静かに保たれる。  

ふつうの日の自己管理は、  
目立たないけれど、いちばん強い。  

波がないときに体勢を整えておくと、  
荒れた日にも、すぐ戻れる。  

頑張らなくてもいい。  
大きなことをしなくてもいい。  

穏やかな日を、  
ただ丁寧に過ごすこと。  

それが、  
自分を守るいちばん確かな方法。

# 015_人と関わる日の心得 ― 世界とペースを合わせる  

人と関わる日は、  
少しだけ呼吸を整えてから外に出よう。  

家を出る前に、  
深呼吸をひとつ。  
声を出す前に、  
心の中で「おはよう」と言ってみる。  

それだけで、  
世界との境界がやわらぐ。  

誰かと話す日は、  
「うまく話す」よりも、  
「相手のリズムを聞く」ことを意識してみよう。  

相手が早く話すなら、  
少しだけ間を置く。  

相手が沈黙したら、  
その沈黙に寄り添う。  

言葉よりも、  
呼吸を合わせるように。  

関わることに疲れるのは、  
世界のリズムと自分のリズムが、  
少しずれているだけ。  

無理に合わせなくていい。  
ゆっくり、タイミングを感じ取ればいい。  

人と関わる日の心得は、  
「うまくやる」ことじゃない。  
「ずれを感じ取る」こと。  

ずれを見つけたら、  
一度、深呼吸をして、  
自分のリズムに戻ればいい。  

世界は、  
少しずつ歩幅を合わせてくれる。  
焦らず、  
自分の速度で関わっていけばいい。

# 016_関わったあとの心得 ― 静かな余韻を受け取る  

誰かと会ったあと、  
なぜか少し疲れることがある。  

それは、うまく話せなかったからでも、  
相手に気を使いすぎたからでもない。  

人と関わるとき、  
私たちは、言葉だけじゃなく、  
たくさんの空気や感情を受け取っている。  

だから、心が静かになるのは、  
自然なこと。  

会話のあとに感じる静けさを、  
「虚しさ」ではなく「余韻」として受け取ろう。  

少しの時間、何もせずに座る。  

深呼吸をして、  
今日、自分が使った言葉を思い出してみる。  

うまく伝えられなくてもいい。  

そのとき出せた言葉が、  
いまの自分の呼吸にいちばん近かっただけ。  

人と関わるとは、  
互いに波を送り合うこと。  

波が届けば、  
その分だけ静けさが戻ってくる。  

静かな余韻の中で、  
世界がまた整っていく。
